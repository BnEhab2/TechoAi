# 📸 Techo Mascot Images Folder / مجلد صور شخصية تيكو

ضع جميع صور تيكو الخاصة بك في هذا المجلد لتعرض فوراً وبشكل تلقائي في الموقع!

Place all your official Techo mascot image files in this folder. The website is pre-configured to automatically load and render them with custom glow effects and float animations!
